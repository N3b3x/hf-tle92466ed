Detailed Description

DITHER_CTRL register bit definitions.

Bit Map:
* Bits 15-14: FAST_MEAS   - Fast measurement period
* Bit 13    : DEEP_DITHER - Deep dither feature
* Bits 11-0 : STEP_SIZE   - Dither step size
