Get VDD thresholds (fixed hardware values, not programmable)

VDD thresholds are fixed hardware values that are NOT stored in registers and cannot be read or programmed. Values are from datasheet Table 9 (Electrical characteristics power supply).

Parameters
uv_thresholdOutput: UV threshold in millivolts
ov_thresholdOutput: OV threshold in millivolts
Note
VDD_UV,TH = 3.7-4.5V (using 4.1V mid-range) VDD_OV,TH = 5.5-6.4V (using 5.95V mid-range)

The datasheet provides min-max ranges, not typical values. We use mid-range estimates.

Get VIO thresholds (fixed hardware values, not programmable)

VIO thresholds are fixed hardware values that depend on VIO_SEL bit setting. These thresholds are NOT stored in registers and cannot be read or programmed. Values are from datasheet Table 9 (Electrical characteristics power supply).

Parameters
uv_thresholdOutput: UV threshold in millivolts
ov_thresholdOutput: OV threshold in millivolts
vio_5vVIO_SEL setting (false=3.3V mode, true=5V mode)
Note
  • 3.3V mode (VIO_SEL=0): UV = 2.6-3.0V (using 2.8V), OV = 3.6-4.1V (using 3.85V)
  • 5V mode (VIO_SEL=1): UV = 3.7-4.5V (using 4.1V), OV = 5.5-6.4V (using 5.95V)

The datasheet provides min-max ranges, not typical values. We use mid-range estimates.
